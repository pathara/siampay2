เอเชียเพย์มีความภูมิใจที่จะประกาศว่าเอเชียเพย์ได้ร่วมมือในการขยายตลาดการชำระเงินออนไลน์ของบัตรเติมเงิน Octopus Cards Limited (OCL) ซึ่งจะนำเสนอทางเลือกในการชำระเงินแก่นักช้อปปิ้งออนไลน์ในฮ่องกง ลูกค้าสามารถเพลิดเพลินไปกับการชำะเงินแบบ OOP ด้วยความปลอดภัยขั้นสูงผ่านทางโทรศัพท์มือถือที่รองรับการใช้งาน NFC (Near Field Communication ) กรุณาคลิก ที่นี่ เพื่อดูบทความเต็มรูปแบบ

10 กุมภาพันธ์ 2558
เอเชียเพย์เปิดตัว Access Control Server (ACS) รูปแบบใหม่สำหรับธนาคาร
เอเชียเพย์เปิดตัว Access Control Server (ACS) รูปแบบใหม่สำหรับธนาคารซึ่งจะช่วยให้ธนาคารผู้ออกบัตร(Visa, Master และ JCB) ช่วยลดปัญหาการขอรับเงินคืนจากการชำระเงินที่ไม่พึงประสงค์ต่างๆ หรือ จากการทุจริตสำหรับการทำธุรกรรมการชำระเงินออนไลน์ ระบบ ACS นั้นมีจุดแข็งอยู่ ตรงการจัดการข้อมูล และ การเข้ารหัสธุรกรรมออนไลน์ในระบบ ซึ่งจะช่วยให้ลดค่าใช้จ่ายของระบบอีกด้วย  กรุณาคลิก ที่นี่ เพื่อดูบทความเต็มรูปแบบ
