16-17 เมษายน 2561
เอเชียเพย์เข้าร่วมงาน Visa Asia Pacific Security Summit 2018 in Singapore.ที่สิงคโปร์

งานประชุม Visa Asia Pacific Security Summit 2018 จัดขึ้นที่ศูนย์การประชุมสิงคโปร์แรฟเฟิลส์ซิตี้ ในวันที่ 16-17 พฤษภาคมที่ผ่านมา งานประชุมนี้ มีผู้เข้าร่วมประชุมมากกว่า 522 รายซึ่งเป็นผู้เชี่ยวชาญในอุตสาหกรรมและหน่วยงานที่กำกับดูแลกฎหมายทั่วภูมิภาคเอเชียแปซิฟิก วัตถุประสงค์ของการประชุมคือหารือเกี่ยวกับแนวโน้มสำคัญที่ส่งผลต่อระบบอุตสาหกรรมชำระเงิน ในฐานะที่เอเชียเพย์เป็นผู้ให้บริการการชำระเงินออนไลน์ชั้นนำของเอเชีย เราภูมิใจที่ได้เป็นหนึ่งในตัวแทนของวีซ่าซึ่งได้เข้าร่วมการประชุมครั้งนี้เพื่อเรียนรู้เกี่ยวกับผลิตภัณฑ์และความเสี่ยงด้านความปลอดภัยของข้อมูลและการฉ้อโกงในการชำระเงิน นอกจากนี้เรายังได้นำเสนอเทคโนโลยีการชำระเงินที่หลากหลายแก่อุตสาหกรรมการธนาคาร
